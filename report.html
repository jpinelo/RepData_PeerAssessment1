Q1 - What is the mean total number of steps taken per day?

Note: For this part of the assignment, you can ignore the missing values in the dataset.

Requirements:
Q1.1 Calculate the total number of steps taken per day Q1.2. Make a histogram of the total number of steps taken each day.

stepsPerday <- df %>%
  group_by(date) %>%
  summarize(daily.steps = sum(steps, na.rm = TRUE))
